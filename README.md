## 关于这个园子

这里是一个安静的角落。

在这个信息过载的时代，我把自己分成了三份：一部分埋在代码里找结构之美，一部分浸在游戏世界里看另一种叙事，还有一份留给日常的细碎光影。

博客创建于 2025 年，没有订阅推送，没有社交按钮，没有算法推荐。只有文字、图片，和缓慢生长的思考。

## 内容分类

### 技术 · Tech

记录后端、前端、架构设计与工具实践中遇到的真正有价值的思考。不追热点，只写自己验证过的东西。

### 游戏 · Game

不评分数，不写通稿。只记录那些让我停下来想一想的游戏——以及它们的设计哲学。

### 生活 · Life

旅行见闻、读书笔记、远程工作的自由与孤独。生活不需要标题党，真实就够了。

## 技术栈

零依赖、纯静态，用最朴素的方式存在：

- **结构：** `index.html`（框架）+ `pages/`（内容）+ `iframe` 路由
- **样式：** CSS Custom Properties 设计令牌，模块化 CSS 文件
- **脚本：** 不到 200 行的原生 JavaScript，无框架、无构建
- **部署：** GitHub Pages，推送即上线

Enjoy the simplicity.
