Plugins 插件机制

插件是 JS/TS 模块，导出一个返回 hooks 的函数——通过订阅生命周期事件来扩展或改写 OpenCode 行为。两种加载方式，启动时自动加载。

① 本地文件

  • 放 .opencode/plugins/（项目级）
  • 或 ~/.config/opencode/plugins/（全局）
  • 启动时自动加载，无需安装

② npm 包

"plugin": ["opencode-helicone-session"]

Bun 在启动时自动安装，缓存到 ~/.cache/opencode/

// .opencode/plugins/my-plugin.ts
export const MyPlugin = async ({ client, $, directory }) => ({
"tool.execute.before": async (input, output) => {
if (input.tool === "bash") { /* 审查/改写命令 */ }
}
})
