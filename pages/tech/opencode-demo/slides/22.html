Subagents 子代理

OpenCode 的代理分两类：Primary 是你直接对话的主代理（用 Tab 切换），Subagent 是主代理可委派、也可被你 @提及 调用的专门助手。

内置 Primary（2）

  • Build · 默认，全工具开放，正常开发
  • Plan · 受限，只读+分析，不改动代码

内置 Subagent（3）

  • General · 多步研究/并行任务，可改文件
  • Explore · 快速只读探索代码库
  • Scout · 拉取外部依赖到缓存做对照
// opencode.json 自定义子代理
"agent": {
"code-reviewer": {
"model": "anthropic/claude-sonnet-4-5",
"mode": "subagent",
"permission": { "edit": "deny" }
}
}

调用方式

  • 自动委派：主代理按 description 自动选
  • 手动 @：@general 帮我查这个函数
  • 子会话：用 +Down 进入、←/→ 循环
