MCP 模型上下文协议

通过 MCP（Model Context Protocol）把外部工具接进 OpenCode——本地命令行进程或远程 HTTP 端点皆可。接入后，MCP 工具会与内置工具一起自动对 LLM 可用。

// Local 类型
"mcp": {
"filesystem": {
"type": "local",
"command": ["npx","-y","@modelcontextprotocol/server-filesystem","/tmp"],
"timeout": 10000
}
}
// Remote 类型
"mcp": {
"jira": {
"type": "remote",
"url": "https://jira.example.com/mcp",
"headers": { "Authorization": "Bearer ..." }
}
}

靠"名字"调用

提示词里写 use the filesystem tool

enabled 开关

设 false 临时禁用，不删配置

⚠️ Context 成本

工具多会挤占上下文，按需启用
