opencode.json 配置

opencode.json 是 OpenCode 的核心配置文件，用于定义项目级的行为和偏好。

用户级配置

~/.config/opencode/opencode.json

全局默认配置，所有项目共享

项目级配置

项目根目录/.opencode/opencode.json

优先级更高，覆盖用户级配置

配置结构示例

{ "model": "anthropic/claude-opus-4-7", "tools": { "shell": { "enabled": true, "autoApprove": [] }, "file": { "enabled": true } }, "plugins": ["oh-my-opencode"] }

常用配置项

model默认模型
tools工具开关
plugins加载插件
memory记忆配置
