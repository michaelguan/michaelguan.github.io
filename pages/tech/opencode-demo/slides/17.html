AGENTS.md 规范

AGENTS.md 是 OpenCode 的标准化项目规范文件，用于定义智能体在项目中的行为和规则。

这与 Claude Code 共享同一规范，生态互通。

项目规则

  • 编码规范定义
  • 技术栈说明
  • 项目架构概述

工具权限

  • 允许的工具列表
  • 危险操作限制
  • 自动批准规则

典型内容结构

--- name: 项目名称 description: 项目描述 tools: - shell: "允许的 shell 命令" - file: "文件操作规则" rules: - "代码风格要求" - "测试规范要求" ---

位置与优先级

放在项目根目录的 AGENTS.md，OpenCode 启动时自动读取，为该项目提供上下文和行为指导。
