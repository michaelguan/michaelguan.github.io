5. Model Builder (金融建模器)

DCF / LBO / 三张表 / comps — 在 Excel 中直接造出可追踪、可勾稽、可交班的财务模型

输入 Input

  • 目标公司 + 可用的 filings/历史数据
  • 模型类型 (DCF / LBO / 3-statement / comps)
  • 可选模板与核算口径

输出 Output

  • Excel 模型工作簿 — 每单元格为可追踪公式，蓝/黑/绿三色规范
  • balance check 与 audit-xls 报告
  • 敏感性/情景表 (可选)

适用场景

分析师立模型起点 — DCF、LBO、三张表、对标估值

技能清单 Skills

dcf-model lbo-model 3-statement-model comps-analysis audit-xls xlsx-author
