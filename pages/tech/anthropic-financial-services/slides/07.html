四种部署方式

1. Cowork 插件

Settings → Plugins → Add plugin
Paste: github.com/anthropics/...
选择 agents

适用：团队随取随用，无需自建

2. Claude Code CLI

claude plugin marketplace add anthropics/financial-services
claude plugin install financial-analysis@...

适用：开发者本地集成

3. Managed Agents API

export ANTHROPIC_API_KEY=sk-ant-...
scripts/deploy-managed-agent.sh gl-reconciler

适用：企业工作流自托管

4. Microsoft 365 Add-in

claude plugin install claude-for-msft-365-install
/claude-for-msft-365-install:setup

适用：Excel/PPT/Word 内嵌

内网落地三原则

能封装不改写

优先封装行内系统接口为 MCP，不动 Agent 代码

能静态不实时

非实时场景用 Excel/CSV 静态注入，减少对接

能分批不一步到位

先上不依赖外部数据的 Agent，再有替代方案者

Agent 是效率工具而非决策者 — 涉及金额、合规、对外输出必须 Human-in-the-Loop

同一系统提示、同一技能集—跨形态一致
