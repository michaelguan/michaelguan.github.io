8. Month-End Closer (月结闭环)

把月结三件事 — 应计、滚动、差异评论 — 串成一条可复核的闭环

输入 Input

  • 当月的总账试算 / 期初余额
  • 应计政策与触发条件
  • 上月数字与预算

输出 Output

  • accrual-schedule 应计表 (生自动计提，留政策依据)
  • roll-forward 滚动表 (期初→本期动→期末)
  • variance-commentary 差异评论稿 (与预算/上年同期)

适用场景 When to use

月末关账;基金运营 / 公司财务的月末结账冲刺

技能清单 Skills

accrual-schedule roll-forward variance-commentary audit-xls xlsx-author
